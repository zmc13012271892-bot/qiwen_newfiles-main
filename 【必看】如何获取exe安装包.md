# 获取启文 .exe 安装包 · 傻瓜式操作说明

## 为什么要这样做？

把代码变成 .exe 需要"编译"，这个过程需要一台有完整环境的电脑运行约 10 分钟。
GitHub 提供免费的云电脑帮你完成这件事，你只需要上传代码，它自动编译好 .exe 给你下载。

全程大约 15 分钟，完全免费。

---

## 第一步：注册 GitHub 账号（已有可跳过）

1. 打开浏览器，访问：https://github.com
2. 点右上角 Sign up
3. 填写邮箱、用户名、密码，完成注册

---

## 第二步：创建仓库

1. 登录后点右上角 "+" 号 → New repository
2. Repository name 填：qiwen
3. 选 Private（私有，别人看不到你的代码）
4. 点绿色按钮 Create repository

---

## 第三步：上传代码

1. 在新建的仓库页面，点 uploading an existing file 链接
2. 把你解压的 qiwen 文件夹里的【所有文件】拖入页面
   ⚠️ 注意：是文件夹里面的文件，不是文件夹本身
   ⚠️ 必须包含 .github 文件夹（可能是隐藏的，Windows 需要勾选"显示隐藏文件"）
3. 拖完后滚到最下面，点绿色 Commit changes 按钮

---

## 第四步：等待自动编译

1. 上传完成后，点页面顶部的 Actions 标签
2. 能看到"编译启文安装包"正在运行（黄色圆圈转动）
3. 等待约 10-15 分钟
4. 变成绿色勾勾说明编译成功！

---

## 第五步：下载 EXE

1. 点那个绿色的工作流记录
2. 滚到页面最下方，找到 Artifacts 区域
3. 点击"启文-Windows安装包"下载
4. 解压下载的 zip，里面就是 .exe 安装程序！

---

## 双击 EXE 就能安装启文了！

安装完成后桌面会出现启文图标，双击打开就是你的软件。

---

## 遇到问题？

Actions 页面如果显示红色 X：
1. 点进去看错误信息
2. 把错误信息截图发给我，我帮你解决
