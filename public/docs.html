使用文档

欢迎使用启文！这份文档将帮助你快速上手，充分发挥启文的所有功能。

快速开始

启文是一款本地优先的知识管理与写作平台，支持 Windows、macOS 和 Linux。下载安装后即可使用，无需注册账户（注册可开启云同步功能）。

提示：启文完全免费，所有功能无需付费即可使用。

安装说明

Windows

下载 .exe 安装包，双击运行，按照安装向导完成安装。安装完成后可在桌面或开始菜单找到启文图标。

macOS

下载 .dmg 文件，打开后将启文图标拖入应用程序文件夹。首次打开时系统可能提示安全警告，前往「系统设置 → 隐私与安全性」中允许即可。

Linux

下载 .AppImage 文件，赋予执行权限后直接运行：

chmod +x QiWen-*.AppImage
./QiWen-*.AppImage

创建工作区

工作区是启文中组织文档的顶层容器。首次启动时，引导向导会帮你创建第一个工作区。你可以为不同项目创建多个工作区，比如「论文研究」「读书笔记」「工作文档」等。

Markdown 编辑

启文内置基于 CodeMirror 6 的 Markdown 编辑器，支持实时语法高亮。

常用语法

# 一级标题
## 二级标题

**加粗文字**  *斜体文字*

- 无序列表项
1. 有序列表项

> 引用块

`行内代码`

[链接文字](https://example.com)

专注模式

按 Ctrl+Shift+F（Mac：Cmd+Shift+F）进入专注模式，隐藏侧边栏和右侧面板，全身心投入写作。

文献管理

启文内置专业的文献管理系统，支持从多种来源导入文献。

导入方式

  • DOI 导入：在文献库中点击「+」，输入 DOI 号自动获取元数据
  • URL 导入：粘贴网页链接，自动解析标题、作者等信息
  • BibTeX 导入：从 Zotero、Endnote 等软件导出 .bib 文件后导入
  • PDF 导入：拖入 PDF 文件，自动识别文献信息

AI 写作助手

在编辑器中唤起右侧「AI」面板，或选中文字后右键选择 AI 操作：

  • 继续写作：根据上下文智能续写
  • 改写润色：提升语言表达质量
  • 提炼摘要：生成结构化摘要
  • 补充引用：匹配相关文献并插入引用

云同步

注册并登录启文账户后，在「设置 → 账户」中开启云同步。开启后，文档将在后台自动同步，无需手动操作。

注意：云同步需要注册账户，但所有本地功能无需注册即可使用。

导出文档

通过菜单「文件 → 导出」或快捷键导出当前文档，支持格式：

  • PDF（保留排版样式）
  • Word（.docx）
  • Markdown（纯文本）
  • HTML（网页格式）

键盘快捷键

Ctrl+N        新建文档
Ctrl+S        保存
Ctrl+K        全局搜索
Ctrl+\        切换侧边栏
Ctrl+Shift+F  专注模式
Ctrl+B        加粗
Ctrl+I        斜体
Ctrl+Z        撤销
Ctrl+Shift+Z  重做

Mac 用户将 Ctrl 替换为 Cmd

插件系统

启文支持第三方插件扩展功能。在「插件市场」中浏览并安装插件，目前支持的插件类型包括：主题样式、导出格式扩展、AI 模型接入、特殊语法支持等。

如需开发插件，请参考 GitHub 上的插件开发文档。
