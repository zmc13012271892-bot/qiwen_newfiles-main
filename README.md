# 启文软件 · 完整使用说明
# 启于思，行于文

===================================================
  看这里！三步让软件跑起来：
  第一步：安装 Node.js
  第二步：双击 一键启动.bat（Windows）
  第三步：等待自动打开
===================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一步：安装 Node.js（只需要装一次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开浏览器，访问：https://nodejs.org
2. 点击绿色大按钮 "LTS" 下载（推荐版本）
3. 下载完成后双击安装，一路点"下一步/Next"即可
4. 安装完成后重启电脑

怎么确认装好了？
  - Windows：按 Win+R，输入 cmd，回车
  - 在黑色窗口里输入：node -v
  - 如果显示版本号（如 v20.0.0）就说明装好了


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二步：启动软件（Windows 用户）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

找到本文件夹里的：  一键启动.bat
双击它！等待 2-5 分钟，软件会自动打开。

第一次启动会自动下载依赖包（需要联网），之后每次秒开。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二步：启动软件（Mac 用户）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

找到本文件夹里的：  一键启动Mac.sh
右键 → 用终端打开

或者打开终端，拖拽 一键启动Mac.sh 到终端窗口，回车。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
打包成安装包（可选，想分发给别人用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows 用户双击：  打包Windows安装包.bat
Mac 用户运行：      bash 打包Mac安装包.sh

打包完成后，dist/ 文件夹里就是安装包文件。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
遇到问题怎么办？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问题1：双击 bat 闪一下就消失了
→ 说明 Node.js 没装好，重新装一遍

问题2：提示"npm 不是内部命令"
→ 重启电脑后再试

问题3：下载依赖很慢
→ 等待即可，国内网络第一次可能需要 5-10 分钟

问题4：软件打开后是白屏
→ 关闭后重新双击 bat 再试

问题5：提示安全警告（Windows Defender）
→ 点"更多信息" → "仍要运行"
